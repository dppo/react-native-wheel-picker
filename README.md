# react-native-wheel-picker

react-native wheel picker

## Installation

```sh
npm install react-native-wheel-picker
```

## Usage

```js
import WheelPicker from "react-native-wheel-picker";

// ...

const result = await WheelPicker.multiply(3, 7);
```
